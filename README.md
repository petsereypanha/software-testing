## 🚀 Quick Start for Interns

### 1. Set Up Your Accounts (Day 1)
- [ ] **Jira** — [atlassian.com/software/jira](https://www.atlassian.com/software/jira) (free tier)
- [ ] **GitHub** — [github.com](https://github.com) (create account)
- [ ] **Postman** — [postman.com](https://www.postman.com) (free account)
- [ ] **Notion or Confluence** — for notes and documentation
- [ ] **Slack** — team communication

### 2. Install Core Tools (Week 1)
- [ ] **Google Chrome** + DevTools (built-in)
- [ ] **draw.io** — [diagrams.net](https://www.diagrams.net) (browser-based, free)
- [ ] **Loom** — [loom.com](https://loom.com) (free screen recording)

### 3. Install Automation Tools (Week 8)
```bash
# Python 3.x
python --version

# Selenium
pip install selenium pytest pytest-html

# ChromeDriver (managed automatically with selenium >= 4.6)
```
